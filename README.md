# TerrainDestruction

## Le projet

Projet de génération procédural basé sur l'algorithme marching cubes avec terrain destructible, qui inclut:
- Génération d'un mesh procédural via l'algorithme marching cubes basé sur une height map 3D
- Outil de destruction de terrain qui modifie le mesh
- Sauvegarde du terrain détruit
- Monde infinie
- Génération avec chunk

Pour reset la sauvegarde suprimer le dossier save dans le projet.

## Commande

Déplacement: ZQSD

Monté: A

Décendre: E

Détruire: clic gauche souris

Bouger la caméra: Mouvement Souris
